MFCPretest01 – C++/MFC GUI 프로젝트

본 프로젝트는 Visual Studio C++ 및 MFC 기반으로 구현된
간단한 Windows GUI 프로그램으로, 데이터 생성·표시·스레드 테스트 등의 기능을 포함합니다.

*포트폴리오 제출을 위해 리팩토링 및 회사명 제거 버전으로 정리되었습니다.

MFC 기반 Windows GUI 구성

-이미지 출력 영역 구현

-사용자 입력 처리 (원형 반지름, 가장자리 두께 등)

-랜덤 데이터 생성 (Set Random)

-데이터 설정(Set Data) 기능

-멀티스레드 처리(Thread Test)

-UI 초기화(Reset)


*해당 프로젝트는

-C++ 기반 로직 구현

-MFC 기반 UI 구성

-윈도우 메시지 처리 경험

-멀티스레드 동작 테스트

등을 포함하여 Windows C++ 개발자로서 기본 역량을 보여주기 위한 포트폴리오 프로젝트입니다.
